FCC Basejump URL shortener

A 'URL changer'. It makes some URLs shorter, others longer. Try it!

Example usage

https://fcc-urlshort-otmeek.c9users.io/new/https://www.google.com
https://fcc-urlshort-otmeek.c9users.io/new/http://www.freecodecamp.com/

Example output

{ "original_url": "https://www.google.com", "short_url": "https://fcc-urlshort-otmeek.c9users.io/1" }
